# TavolaPro - Restaurant Management SaaS

Complete B2B SaaS for restaurants and pizzerias: menu management, table orders, reservations, inventory, staff scheduling, and advanced reporting.

## What you received
The online site is a demo running with local data. This code is the complete, production-ready application. To make it real, you just need to connect your own services (Supabase, Stripe, SendGrid) by following these instructions.

## Requirements
- Node.js 18+
- A Supabase account (Database & Auth)
- A Stripe account (Payments)
- A SendGrid account (Emails)
